# go-liner
if you type `liner`, this script print `##########`.  
And Copy to Clipboard.  

ex)
- `liner -l 5 ` -> #####
- `liner -l 10` -> ##########

## Usage
### Option
- `-l`  
  This Option is line length. (default 10)
- `-m`  
  Line Char. (default #)   
  ex)  
    `-m #`  -> ##########  
    `-m ?`  -> ??????????  
    `-m ##` -> ####################  
- `-no`  
  No Copy Flag.
